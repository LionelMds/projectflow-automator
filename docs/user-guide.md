# Guide utilisateur

## Premier lancement

1. Ouvrir ProjectFlow Automator.
2. Cliquer sur `Se connecter a Microsoft`.
3. Selectionner les chemins :
   - racine projets,
   - dossier de reference,
   - repertoire chantier Excel.
4. Renseigner le plan et le bucket Planner si les taches doivent etre creees.

## Creer un projet

1. Saisir l'annee et l'ID projet, par exemple `2026` et `4995`.
2. Renseigner la designation et les informations client disponibles.
3. Activer Planner si une tache doit etre creee.
4. Cliquer sur `Creer`.

## Creer un sous-projet

Ajouter le numero de sous-projet dans le champ inline, par exemple `2` pour `2026-4995-2`.

Le sous-projet reutilise le dossier parent. Il cree uniquement une nouvelle fiche et une ligne de repertoire.

## Mettre a jour

Le bouton `Mettre a jour` reecrit la fiche et la ligne du repertoire apres confirmation. Il ne cree pas de dossier, Outlook ni Planner.
